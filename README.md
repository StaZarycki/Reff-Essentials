Reff Essentials for Osu

Po prostu pobierz wersję, która cię interesuje (prawdopodobnie najnowsza).
Nie obrażę się za podanie dalej :3

3. Dostosuj ilość punktów
4. Kliknij "Kopiuj"
5. Teraz wynik możesz wkleić np. do lobby na multi
